# Created Terraform code that includes:
-  2 Load Balancers -  Internet Facing and Internal 
-  4 Web Servers
-  4 Application Servers 
-  RDS with two nodes (MySQL).
-  S3 Bucket.

### To run the project
#### in your terminal
Init 
```
terraform init
```
Plan
```
terraform plan
```
Apply 
```
terraform apply
```
